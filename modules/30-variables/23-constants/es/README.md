
Algunos datos nunca cambian, como las constantes matemáticas. Tomemos como ejemplo el número π. Siempre es igual a 3.14 y no puede cambiar. Para acceder a este tipo de datos, en Python se utilizan constantes:

```python
PI = 3.14
print(PI)  # => 3.14
```

Una constante se crea de la misma manera que una variable. La única diferencia es que las constantes se suelen nombrar con letras mayúsculas y se usa `_` como separador entre palabras. Una constante, al igual que una variable, se puede utilizar en cualquier expresión.
